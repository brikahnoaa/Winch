Switching a platform between CF1 and CF2

 

Programs compiled for CF2 would not run on CF1 platform, or vice versa.  The program environment in CodeWarrior has to agree with the platform it's running.

  

Start CodeWarrior.

Under IDE Switch (within Metrowerks CodeWarrior), click

Edit:Preferences…

In IDE Preference Panels: click General:Source Trees

Select MxTarget, click Choose… button

Navigate to C:\Program Files\Persistor\MotoCross Support\CFX

Select (CF1) or (CF2) and click OK button

Click Change button (expect warnings)

Click OK button

 

In accordance to the system type between CF1 and CF2, comment out one of these lines in cfx.h file under "Header" files

//#define CF1

#define CF2 

 

Recompile the program and test it thoroughly!!!
